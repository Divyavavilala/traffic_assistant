3: Check Firewall Status
Search: Windows Defender Firewall
Click Turn Windows Defender Firewall on or off

4 Check Antivirus (Windows Security)
Open Windows Security
Go to Virus & Threat Protection
Real-time protection → ON
Last scan → Recent

5: Check Password & Account Security
Settings → Accounts → Sign-in options
Password / PIN enabled
Windows Hello (optional)

6: Check Installed Applications
Settings → Apps → Installed apps
Look for:
Unknown software
Cracked tools

7: Check Startup Programs
Press Ctrl + Shift + Esc
Go to Startup tab
Disable: Unknown apps

8: Check Network Security
Settings → Network & Internet
Check:
WiFi connection
Network type → Private/Public

9: Check Browser Security
Steps (Chrome/Edge):
Go to Settings → Privacy & Security
Check:
Safe browsing ON
Remove unknown extensions

10: Check Data Backup
Search: Backup settings
Check:
OneDrive / External backup
